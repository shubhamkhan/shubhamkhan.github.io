Food Shala E-commerce Website Prototype

Project Brief:

This is a small prototype of an e-commerce website. The application has 2 types of users, restaurant and customer. There is a page that displays food items available in restaurants as well as an 'order' button. This page is accessible to everyone, regardless of whether the user is logged in or not. It is implemented cart functionality. And on the ‘Order’ page, restaurants are able to see a list of all the customers who have ordered food from their restaurant along with the food items they have ordered from their restaurant.

  • Date - 2021
  • Client - Self-made project
  • Tools - HTML, Bootstrap, JQuery, AJAX and PHP & MySQL

IoT based UAV for surveillance using WSN

Project Brief:

IoT based Unmanned Aerial Vehicle (UAV) for surveillance using Wireless Sensor Network (WSN) project main goal is to build a video surveillance UAV. Here we can remotely control the UAV by using remote and send the video directly to the base station with the help of the camera mounted on the UAV. To make this project more advanced, we will be tracking human movement using UAV video footage. This project can be used for search and rescue, military application, product delivery, and news reporting.
